What is Packet Switching?

Packet switching is the main computing invention that allowed programmers to create the internet. Packet switching is simply the way some object or data is transfered between two endpoints. The main idea of packet switching in computing is to break up some amout of data into smaller parts or packits and send it accross the network. Each individual packet does not travel across the same route between endpoints, rather each packet travels the best route to take to the endpoint which can change between packets. The packets travel through nodes and the current node decides which next node is best to send the packet to.
